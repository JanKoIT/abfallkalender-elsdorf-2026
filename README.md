## Enthaltene Abfallarten

- **Hausmuell** - 14-tagige Abholung
- **Biotonne** - 14-tagig (Jan-Marz), wochentlich (April-November)
- **Gelbe Tonne** - 14-tagige Abholung (Leichtverpackungen)
- **Blaue Tonne** - Monatliche Abholung (Altpapier)
- **Grunabfall** - 4 Termine pro Jahr
- **Elektroschrott** - 4 Termine pro Jahr
- **Schadstoff** - Sondertermine

## Installation

### Google Calendar
1. Offnen Sie [Google Calendar](https://calendar.google.com)
2. Klicken Sie auf das Zahnrad-Symbol > Einstellungen
3. Wahlen Sie "Importieren & Exportieren"
4. Klicken Sie auf "Datei von Ihrem Computer auswahlen"
5. Wahlen Sie die heruntergeladene .ics-Datei

### Apple Kalender (macOS/iOS)
1. Doppelklicken Sie auf die .ics-Datei
2. Wahlen Sie den Kalender, in den importiert werden soll
3. Klicken Sie auf "Importieren"

### Microsoft Outlook
1. Offnen Sie Outlook
2. Datei > Offnen & Exportieren > Importieren/Exportieren
3. Wahlen Sie "iCalendar-Datei (.ics) importieren"
4. Wahlen Sie die heruntergeladene .ics-Datei
